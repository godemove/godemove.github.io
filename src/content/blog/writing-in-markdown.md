对于写作者来说，最大的敌人往往不是灵感枯竭，而是**格式工具栏**。

Markdown 解决了这个问题。它用简单的符号来表示格式，让你可以始终沉浸在文字的流动中。

## 常用语法

```markdown
# 一级标题
## 二级标题
**粗体**，*斜体*
[链接文字](https://example.com)
`行内代码`
```

## 代码高亮

Astro 原生支持代码块的语法高亮：

```js
const greeting = "Hello, world!";
console.log(greeting);
```

## 写作即思考

当你不需要操心字体、颜色、间距时，你的大脑就能释放出更多认知资源，用于真正重要的事——**思考与表达**。
